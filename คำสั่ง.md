# คำสั่ง Git สำหรับบันทึกความคืบหน้า (SAMS Project)

ใช้คำสั่งนี้ใน Terminal เพื่อทำการบันทึกประวัติการพัฒนาและอัปโหลดโค้ดเวอร์ชันล่าสุดขึ้นไปยัง GitHub:

```bash
git add . && git commit -m "Fix: dashboardService schedule_id bug and update reports" && git push origin main
```

---
*หมายเหตุ: คำสั่งนี้จะบันทึกทุกไฟล์ที่มีการเปลี่ยนแปลงล่าสุดในวันนี้ และส่งขึ้นสู่ GitHub สาขาหลัก (main)*
